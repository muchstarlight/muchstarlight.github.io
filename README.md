# 多新光基地

> 一个高中生随便写写的地方，技术很菜，但坚持很酷。

[**🌐 在线访问**](https://blog.muchstarlight.top)

---

## 这是什么

这是我的个人博客，主要用来堆放一些学习笔记、折腾记录、偶尔的随笔，以及不知道从哪里冒出来的奇怪想法。

我是 **多星光**（muchstarlight），目前还是个高中生，喜欢电脑、手工一类的东西。技术力有限，代码写得不算漂亮，但搭这个博客的过程本身就挺有意思的。

## 博客里有什么

- 📝 **文章** — 学习记录、折腾心得、偶尔的思考碎片
- 💬 **留言板** — 欢迎来唠嗑，用 GitHub Discussions 驱动的评论系统
- 🎵 **音乐** — 侧边栏有个播放器，干活的时候喜欢听点歌
- 💰 **赞助** — 如果我的文章帮到了你，可以在爱发电请我喝杯奶茶（不是必须！）
- 🔗 **友链** — 目前空缺中，欢迎来换（不过请先挂上我哦）

## 怎么搭起来的

这个博客基于 [**Firefly**](https://github.com/CuteLeaf/Firefly) 主题，用的是 [Astro](https://astro.build) 框架。Firefly 是从 [saicaca/fuwari](https://github.com/saicaca/fuwari) Fork 来的，真的很好看，感谢作者们。

部署在 GitHub Pages 上，域名是 `blog.muchstarlight.top`。

## 本地跑起来

```bash
pnpm install
pnpm dev
```

然后打开 `http://localhost:4321`。

## 一些废话

这个博客是一拍脑门建起来的，很多配置还在慢慢完善。内容也不一定严谨，但每篇都是认真写的。如果有什么地方写错了，欢迎直接在评论区指出，<del>我改得很快</del>（看见了大概就会改）。

另外，如果哪天我突发奇想，这里可能会大变样。谁知道呢。
